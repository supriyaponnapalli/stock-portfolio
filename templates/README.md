# Stock Portfolio Tracker

A simple web application to track your stock investments.

## Features
- Add stocks with symbol, quantity and buy price
- View portfolio summary (Total Invested, Current Value, Profit/Loss)
- Clean and responsive design
- Built with Python Flask

## Tech Stack
- Python + Flask
- HTML, CSS, JavaScript
- Jinja2 Templating

## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/supriyaponnapalli/stock-portfolio.git

# Go to project folder
cd stock-portfolio

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
